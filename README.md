# Power-BI-Projects

Welcome to my **Power BI Projects** repository!  

This folder contains my collection of **Power BI projects**, created as part of my learning journey in **data analytics**. Each project focuses on improving skills in:

- **Data Cleaning** – Preparing raw data for analysis.  
- **Data Modeling** – Building relationships and structured datasets.  
- **DAX (Data Analysis Expressions)** – Performing advanced calculations.  
- **Data Visualization** – Creating interactive and insightful dashboards.  
- **Storytelling** – Presenting data insights effectively.  

## Projects

Each project is organized in its own subfolder with:

- **Power BI file (.pbix)**  
- **Data source(s)**  
- **Supporting documentation** (if any)  

## Purpose

The goal of these projects is to:

- Practice and improve my Power BI skills  
- Apply analytics concepts to real-world datasets  
- Build a portfolio to showcase my learning and capabilities
